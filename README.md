# CarpoolConnect

## Descripción del Proyecto

**CarpoolConnect** es una aplicación móvil híbrida diseñada para facilitar el carpooling (viajes compartidos) de manera segura, eficiente y sostenible. Su objetivo principal es conectar a conductores y pasajeros que realizan trayectos similares, permitiendo la coordinación de viajes grupales de forma sencilla e intuitiva.

## Objetivos Principales

- Conectar usuarios con rutas similares.
- Reducir gastos de transporte mediante viajes compartidos.
- Fomentar la movilidad sostenible y colaborativa.
